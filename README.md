# Task 8: Clustering with K-Means

## 📌 Objective
To perform unsupervised learning using K-Means clustering on the Mall Customers dataset and identify meaningful customer segments.

## 🛠 Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📂 Dataset
Mall Customers Dataset  
Features used:
- Annual Income (k$)
- Spending Score (1-100)

## 🚀 Steps Performed
1. Loaded and explored the dataset.
2. Selected relevant features for clustering.
3. Applied StandardScaler for feature scaling.
4. Used the Elbow Method to determine optimal number of clusters.
5. Applied K-Means clustering (K = 5).
6. Evaluated model using Silhouette Score.
7. Visualized clusters using scatter plots and PCA.

## 📊 Results
- Optimal Clusters (K): 5
- Silhouette Score: 0.55
- Clear separation of customer segments observed.

## 📈 Key Learning
- Understanding unsupervised learning concepts.
- How K-Means clustering works.
- Importance of choosing optimal K.
- Cluster evaluation using Silhouette Score.

## 🧠 Conclusion
The K-Means algorithm successfully segmented customers into 5 meaningful groups. The clustering quality is good based on the Silhouette Score, making it useful for customer segmentation and targeted marketing strategies.
